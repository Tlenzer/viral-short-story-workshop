# 短篇骨架与走向表

大多数平台型短篇，正文节奏共用同一套骨架。
真正变化的，主要是入口和走向。

这份参考只保留可复用结构，不按具体作品分类。

## 一、四类开局骨架

### 1. 情绪受害型

适用：

- 家庭
- 婚恋
- 校园
- 职场
- 成长冲突

骨架：

1. 主角先吃亏或被压
2. 对方做出突破底线的事
3. 主角不再忍
4. 第一轮反击出现
5. 对方继续作死
6. 卡在更大翻盘前

### 2. 认知错位型

适用：

- 规则异常
- 末世
- 悬疑
- 脑洞
- 修仙异能

骨架：

1. 抛出反常事实
2. 外界按错误规则行事
3. 主角先察觉不对
4. 主角试探或验证
5. 更大的异常出现
6. 卡在真相彻底揭开前

### 3. 外部声音介入型

适用：

- 弹幕
- 热帖
- 评论区
- 系统提示
- 围观舆论

骨架：

1. 外部声音先下判断
2. 现实局面和判断形成错位
3. 主角被误判、被裹挟或先得知秘密
4. 读者完成站队
5. 评论和现实互相打脸
6. 卡在舆论翻面前

### 4. 身份反差型

适用：

- 主角身份本身有记忆点
- 危险身份配反差性格
- 被误解身份配隐藏能力

骨架：

1. 先亮身份
2. 立刻补反差
3. 反差被卷入高压事件
4. 主角开始动作
5. 外界重新评估主角
6. 卡在身份彻底揭牌前

## 二、五类剧情走向

### 1. 打脸反杀

节奏重点：

- 让压迫越来越具体
- 让反击越来越有代价
- 让对方翻车越来越公开

### 2. 误会反转

节奏重点：

- 先让读者接受一种判断
- 再用新信息逐层翻面
- 每次反转都要改写立场

### 3. 身份揭露

节奏重点：

- 先压信息
- 再埋能回收的线索
- 最后在值钱节点揭牌

### 4. 规则解谜

节奏重点：

- 先立异常
- 再给错误答案
- 主角通过试错逐渐逼近真规则

### 5. 围观翻车

节奏重点：

- 先形成舆论
- 再让舆论错判
- 最后让舆论反噬错误的一方

## 三、万用 8 节拍

无论题材和走向如何，先把正文拆成这 8 个节点：

1. 开场炸点
2. 关系和站位
3. 对方踩线或局面失控
4. 主角态度变化
5. 第一层底牌或第一次反击
6. 对方继续作死或误会加深
7. 第二层反转、揭牌或更大真相
8. 断点收尾

## 四、最稳的断点位置

1. 证据出现了，但还没公开
2. 舆论开始翻了，但还没彻底翻
3. 真相只揭了一半
4. 大人物刚到场，还没表态
5. 主角刚笑了，还没出手
6. 对方刚意识到自己搞错了

## 五、爽点与追读按钮

- 我本来还能忍，但你踩到了不能踩的线
- 你们都以为我没有底牌
- 你们按错误规则压我，我用真规则压回去
- 你们越觉得我不行，翻面越值钱
- 你们站错队，后面就会更想看翻车
- 我不只要赢，我还要让局面换边

## 六、结构避坑

- 不要同一种冲突循环三遍
- 不要主角只挨打不动作
- 不要支线比主线更热闹
- 不要反转只换说法，不改局面
- 不要结尾把最值钱的信息一次性说光
